# Scania Smart Dash+ 150% Larger Text

### [Subscribe on the Steam Workshop&nbsp;↗](https://steamcommunity.com/sharedfiles/filedetails/?id=3737411431)

![150% larger](https://github.com/notsidney/ets2-mod-scania-smart-dash-plus/raw/main/images/steam_workshop.jpg)

A Euro Truck Simulator 2 mod that increases the size of text and icons in the Scania Smart Dash by 150%.

Additions:

- Added low beam icon
- Changed cruise control text to white for improved readability
- Added speed limit to GPS module
- Added mandatory rest break time to GPS module
- Added ETA to GPS module
- Added clock to GPS module
